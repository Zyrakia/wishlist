## About Wishii

Wishii is a wishlist management platform that helps you organize your wishes and share them with family and friends. Create lists, connect external wishlists, form groups, and coordinate gift-giving without the awkward "who's buying what?" conversations.

## Creator

Wishii was created by **Zyrakia**.

## Open Source

Wishii is open source! You can view the source code, report issues, or contribute on GitHub:

**GitHub:** [github.com/zyrakia/wishlist](https://github.com/zyrakia/wishlist)

## Technology Stack

Wishii is built with modern web technologies:

- **SvelteKit** - Full-stack web framework
- **Svelte 5** - Frontend UI with runes for reactivity
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first styling
- **Drizzle ORM** - Database toolkit
- **LibSQL** - Database
- **Zod** - Schema validation
